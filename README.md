### **1. Введение**

#### **1.1. Актуальность проекта**
В условиях роста киберугроз и сложности сетевой инфраструктуры возрастает потребность в инструментах для оперативного анализа уязвимостей сетевых ресурсов. **MegaPortScanner** разработан как многофункциональный инструмент для сканирования портов, анализа сетевых интерфейсов, сбора баннеров и автоматизации задач безопасности в рамках подхода **Red Team**.

#### **1.2. Цель и задачи проекта**
**Цель:** разработка кроссплатформенного инструмента для анализа сетевой безопасности с графическим интерфейсом и расширенным функционалом сканирования.

**Задачи:**
- Реализация многопоточного сканирования портов.
- Интеграция графического интерфейса на базе Tkinter.
- Добавление модулей для анализа Wi-Fi, работы с историей сканирований и генерации отчетов.
- Обеспечение конфигурируемости и безопасности доступа.

#### **1.3. Обзор предметной области**
Проект относится к области **информационной безопасности**, в частности к **сканированию сетей и анализу уязвимостей**. Аналогичные инструменты: Nmap, Masscan, Zenmap.

### **3. Функциональное описание**

#### **3.1. Основные модули и их назначение**
- **MegaPortScanner** – главный класс приложения.
- **Методы сканирования:**
  - `scan_network_dialog()` – диалог выбора сети.
  - `scan_local_network()` – сканирование локальной сети.
  - `check_single_port()` – проверка одного порта.
- **Модуль анализа:**
  - `analyze_headers_tool()` – анализ заголовков.
  - `get_banner()` – сбор баннеров служб.
- **Утилиты:**
  - `export_history_csv()` – экспорт истории.
  - `generate_report()` – генерация отчетов.
  - `backup_config()` – резервное копирование.

#### **3.2. Графический интерфейс пользователя (GUI)**
Интерфейс включает:
- Вкладки: **Scan**, **Network**, **Tools**, **History**, **Logs**.
- Панель управления сканированием.
- Дерево результатов.
- Встроенный браузер для открытия служб.

#### **3.3. Возможности сканирования и анализа**
- Сканирование диапазонов портов.
- Определение служб и версий ПО.
- Анализ сетевых интерфейсов.
- Сбор истории сканирований.
- Экспорт данных в CSV/JSON.

### **5. Инструкция по развертыванию и использованию**

#### **5.1. Требования к системе**
- ОС: Windows 10+, Linux, macOS.
- Python 3.14+.
- Права администратора для сканирования некоторых портов.

#### **5.2. Установка и запуск**
```bash
git clone [репозиторий]
cd port_scanner
pip install -r requirements.txt
python main.py
```
Или запуск собранного `.exe` из папки `dist/`.

#### **5.3. Примеры использования**
1. Запустить приложение.
2. Выбрать диапазон портов.
3. Указать целевой хост.
4. Начать сканирование.
5. Просмотреть результаты во вкладке **History**.
